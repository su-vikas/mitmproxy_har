mitmproxy_har
=============

A simple script converting mitmproxy's netstring dump to HAR format. 

Written for v0.11 of libmproxy. 

To read more about HAR format visit (https://dvcs.w3.org/hg/webperf/raw-file/tip/specs/HAR/Overview.html)
